Backend Startup
1. Open terminal and navigate to cd "KBK-Picability-Backend"
2. Type "dotnet run" !!DO NOT CLOSE THIS TERMINAL OR DB CONNECTION WILL BE LOST!!

Frontend Startup
1. Open terminal and navigate cd "KBK-Picability-Frontend"
2. Type "npm install"
3. After install, type "npm run dev" !!DO NOT CLOSE THIS TERMINAL OR UI WILL NOT LOAD!!
4. Click the link that pops up in terminal
