Project Setup & Run Steps

Backend Startup (May take a few minutes for server to respond)
1. Open terminal and navigate to cd "KBK-Picability-Backend"
2. Type "dotnet run" !!DO NOT CLOSE THIS TERMINAL OR SERVER CONNECTION WILL STOP!!

Frontend Startup
1. Open terminal and navigate cd "KBK-Picability-Frontend"
2. Type "npm install"
3. After install, type "run npm dev" !!DO NOT CLOSE THIS TERMINAL OR UI WILL NOT LOAD!!
4. Click the link that pops up in terminal
